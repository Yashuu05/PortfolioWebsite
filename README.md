# Portfolio Website 🚀
- A clean, modern, and responsive personal portfolio website designed to showcase projects, skills, and achievements. 
- Built with HTML5, CSS3, and Vanilla JavaScript, this project focuses on performance and ease of customization.

---

# ✨ Features

1. Dynamic Content System: All website content (projects, bio, skills, competitions) is stored in a single JavaScript object (portfolioData), making updates effortless without touching HTML structure.
2. Responsive Design: Fully adaptable layout that looks great on desktops, tablets, and mobile devices.
3. Dark Mode Theme: A professional dark navy and teal color palette (#0a192f, #64ffda).
4. Scroll Animations: Smooth scroll-reveal effects using the Intersection Observer API.
5. Interactive Elements:
6. Hover effects on project cards and buttons.
7. Skill bars with animation.
8. Zoomable Certificate Viewer: Click on certificate thumbnails to view them in a full-screen modal.
9. Competitions Section: Dedicated section for hackathons and contests with team size, host, and contribution details.

---

# 🛠️ Tech Stack

1. Frontend: HTML5, CSS3, JavaScript (ES6+)
2. Icons: FontAwesome
3. Fonts: Montserrat & Inter (Google Fonts)
